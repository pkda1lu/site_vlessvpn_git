Инструкции по настройке

Для наглядности вы можете загрузить скриншоты настроек в папку `assets/img/` и вставить их в эти блоки как изображения.

Android — v2rayNG (VLESS)
  1. Установите v2rayNG из Google Play или APK.
  2. Скопируйте VLESS‑ссылку из Telegram‑бота.
  3. В приложении нажмите «+» → «Добавить из буфера обмена».
  4. Сохраните и включите подключение.
iOS — Streisand / Shadowrocket (VLESS)
  1. Установите клиент из App Store.
  2. Скопируйте VLESS‑ссылку из бота.
  3. Импортируйте конфиг по ссылке.
  4. Разрешите добавление VPN‑профиля и подключитесь.
Windows — v2rayN (VLESS)
  1. Скачайте v2rayN с GitHub Releases.
  2. Добавьте сервер через «Добавить [VLESS] из буфера обмена».
  3. Выберите маршрут «Полный» и нажмите «Запустить сервис».
macOS — V2RayXS (VLESS)
  1. Установите V2RayXS через Homebrew или DMG.
  2. Импортируйте VLESS‑ссылку.
  3. Активируйте подключение из строки меню.
Outline (Android/iOS/desktop)
  1. Установите приложение Outline с официального сайта.
  2. Вставьте ключ доступа из бота «Add key».
  3. Подключитесь одним нажатием.
Если возникли сложности — напишите нам в поддержку.
